# Logo Usage Rule

Beacon is part of Applied AI Solutions and must use AAS logo assets. Do not use placeholder single-letter marks.

Required assets:
- `public/aas-icon-dark.svg`
- `public/aas-logo-glow.svg`

Before preview/deploy:
- no `<div class="mark">A</div>` placeholder logo
- no `<div class="logoMark">A</div>` placeholder logo
- no Applied Irrigation logo bleed
- no broken local image references in preview
